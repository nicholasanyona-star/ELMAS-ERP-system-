# Warehouse Controller Dashboard

## Overview
The Warehouse Controller Dashboard provides warehouse managers with a comprehensive view of inventory status, order fulfillment, and warehouse efficiency.

## Key Features
- **Inventory Tracking**: View real-time updates on stock levels across various locations.
- **Order Management**: Track incoming and outgoing orders, including their status.
- **Performance Metrics**: Analyze warehouse performance with key indicators.

## User Guide
1. **Navigation**: Use the sidebar to access different sections of the dashboard.
2. **Filtering Data**: Apply filters to view specific inventory categories or order statuses.
3. **Generating Reports**: Export performance reports in various formats.
